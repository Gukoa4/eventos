Temario de eventos js

    Eventos

    Definicion
    Un evento es cualquier cambio que ocurre en la pagina
    Event Handlers
    maneja los eventos por ejemplo onclick pero no se utiliza mas
    Event listeners
    Este se usa para escuchar eventos y manipularlos por ejemplo button.addEventListener("click",funcion)
    Objeto Evento
    para verlo se hace boton("click",(evt)=>{console.log(evt)}) y asi muestra las propiedades e.target muestra el elemento que ejecuto el evento
    Event Flow
    Event Bubbling vs Event Capturing
    Bubbling es por defecto y genera que se ejecuten los eventos en orden desde los hijos hacia los padres ademas se puede cambiar el orden agregando "true" para espeificar el que se va a mostrar primero boton("click",(evt)=>{console.log(evt), true})
    event.stopPropagation()
    con esto evitamos que el evento siga, ya no se ejecutan los padres. evt.stopPropragation()

    Eventos de la interfaz

    Error
    Ocurre cuando sucede un error durante la carga
    Load
    Ocurre cuando se ha cargado
    Beforeunload
    Ocurre antes de que el documento este a punto de descargarse
    unload
    Ocurre cada vez que se ha descargado la pagina
    resize
    Ocurre cuando se cambia el tama√±o de la vista del documento
    scroll
    Ocurre cuando se desplaza la barra
    select
    Ocurre despues de que el usuario selecciona un texto del input o text area

    Temporizadores

    setTimeout()
    Es para ejecutar una funcion con delay. Por ejemplo setTimeout (saludar,2000)
    setInterval()
    Es para ejecutar repetidas veces en el intervalo elegido.
    clearTimeout()
    con esto se eliminan los temporizadores que se pasen por parametro. clearTimeout(temporizador,tiempo)
